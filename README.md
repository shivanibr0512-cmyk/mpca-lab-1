Question A:
9. A. Scenario: Alien Pizza Destiny (6 Marks)
An alien machine finds a destiny number for every pizza order.
Starting with an integer order number:
● Add all its digits.
Repeat until a single digit remains.
Given Data
OrderNumber = 987654
Problem Statement
Write a RISC-V assembly program using RARS to:
1. Compute the final single-digit destiny number using repeated digit reduction.
2. Using the destiny number, calculate the factorial of the digit.
Constraints
Use loops and integer arithmetic only. Do not convert the number to a string.
No mathematical shortcuts. The program must run correctly in RARS.

Question B:
B. Scenario: Lift Movement Simulator (4 Marks)
A lift starts at floor 0 and receives a sequence of movement requests.
Problem Statement
Write a RISC-V assembly program using RARS to:
- Calculate final floor position
- Calculate total floors traveled
Constraints
• Use arithmetic instructions only
• Avoid pseudo instructions beyond standard RISC-V
• Program must execute correctly in RARS
